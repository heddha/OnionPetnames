Attention: normally, opening a file via webextension closes the webextension window. We can then not use the file anymore. This is not a desired behaviour, as stated here and here . The team is working on a fix. In order to import things with this extension you need to open the about:config tab and set "ui.popup.disable_autohide" to "true", until the issues have been fixed.
